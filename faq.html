Frequently Asked Questions

Who is this suitable for?

This course will be suitable for you if you:

  1. Are looking for opportunities to get started with ML.
  2. Are somewhat familiar with python (you don't need to be an expert).
  3. Have a basic knowledge of linear algebra and calculus.

What is Explore ML?

Explore Machine Learning (ML) is a Google-sponsored program for university students to get started with Machine Learning. This program is intended for those who wish to learn ML from a practical, applied perspective that will enable them to use machine learning in their projects, and who wish to get started with the basics of TensorFlow.
